Core Components

  • Frontend: React with TypeScript & Vanilla CSS for a fast, modern PWA experience.
  • Database: SQLite WASM with Origin Private File System (OPFS) for persistent, high-performance local storage.
  • Data Fetching: Client-side fetches to Yahoo Finance (via a CORS proxy) to gather daily price movements and fundamental data.
  • Storage Strategy: "Snapshot" model — you trigger a sync, the app fetches data and updates the local SQLite database.
