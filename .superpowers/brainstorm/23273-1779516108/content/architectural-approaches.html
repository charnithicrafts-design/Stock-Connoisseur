Architectural Approaches

Choose the foundation for your Stock Connoisseur.

1. The Local Minimalist

Recommended

A pure PWA using SQLite WASM. All data stays on your device. Fetches yfinance data via a simple CORS proxy.

  • Zero server costs
  • Maximum Privacy
  • Works offline
  • Emails must be manually triggered
  • Data lost if browser cache cleared (without export)

2. The Edge Hybrid

Uses a small Edge Function (Vercel/Cloudflare) to reliably fetch data and potentially send automated emails.

  • More reliable API calls
  • Automated daily emails
  • Backup/Sync options
  • Slightly more complex setup
  • Potential (minimal) API/Function costs

3. The Snapshot Specialist

A GitHub Action runs once a day to fetch data and generate a static "Snapshot" file that the PWA loads.

  • Lightning fast load times
  • Consistent daily updates
  • No runtime API calls
  • New stocks only update on next build
  • Requires GitHub account
